# 📦 Warehouse & Inventory Management System

## 📌 Project Description

This project is built using SQL to manage warehouse inventory, products, suppliers, sales, and purchases.


## 🧠 Features

- Product Management  

- Supplier Management  

- Inventory Tracking  

- Sales Management  

- Purchase Tracking  

- Stored Procedures  

- Triggers  

- Views  

- Window Functions  

## 🗃️ Tables Used

- suppliers  

- products  

- warehouses  

- inventory  

- sales  

- customers  


## ⚙️ SQL Concepts Used

- SELECT, INSERT, UPDATE, DELETE  

- WHERE, LIKE, BETWEEN  

- GROUP BY, HAVING  

- JOINS (INNER, LEFT, RIGHT)  

- Subqueries  

- Views  

- Triggers  

- Stored Procedures  

- Window Functions  

 
## 🚀 How to Run

1. Open MySQL Workbench  

2. Import `warehouse_inventory.sql` file  

3. Run all queries
